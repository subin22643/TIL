class Person:    # class는 본인의 속성을 가진 객체(iu)를 찍어내고 행동은 인스턴스가 함
	pass

# 인스턴스 생성

# 메서드 호출
iu.메서드()    # Person이 가진 메서드를 사용

iu.attribute    # Person이 가진 속성을 사용

```python
# 클래스 정의
class Person:
    # 속성(변수)
    blood_color = 'red'

    # 메서드
    def __init__(self, name):    # 생성자 메서드, 개별자가 직접 호출X 인스턴스를 만들때 자동으로 호출됨
        self.name = name         # init : 초기화/ self : 미래에 생성될 이름에 바인딩
																 # self는 singer1, name은 iu
    def singing(self):
        return f'{self.name}가 노래합니다.'
    

# 인스턴스 생성
singer1 = Person('iu')    #
singer2 = Person('BTS')

    
# 메서드 호출
print(singer1.singing())    # iu가 노래합니다.
print(singer2.singing())    # BTS가 노래합니다.

# 속성(변수) 사용
print(singer1.blood_color)    # red
print(singer2.blood_color)    # red
```

## 클래스 변수

class Person:
		count = 0

		def __init__(self, name):    # 모든 인스턴스는 기본적으로 name 인스턴스 변수를 가짐 
				self.name = name
				Person.count += 1

Person1 = Person('iu')
Person2 = Person('BTS')

print(Person.count)    # 2

- 인스턴스 메서드        # 인스턴스.메서드
- 클래스 메서드
- 정적 메서드 (스태틱 메서드)

### 인스턴스 메서드

클래스에 의해 생성된 객체들

클래스로부터 생성된 각 인스턴스에서 호출할 수 있는 메서드

>> 인스턴스의 상태를 조작하거나 동작을 수행

### 인스턴스 메서드 구조

- 클래스 내부에 정의되는 메서드의 기본
- 반드시 첫 번째 매개변수로 **인스턴스 자신(self)**을 받음, 없으면 동작 안됨 (self 글자는 바꿔도됨)

```python
class MyClass:
	
	def instance_method(self, arg1, ...):
		pass
```

### 생성자 메서드

인스턴스 객체가 생성될 때 자동으로 호출되는 메서드

인스턴스 변수들의 초기값을 설정

### 클래스 메서드

클래스가 호출하는 메서드

클래스 변수를 조작하거나 클래스 레벨의 동작을 수행

### 클래스 메서드 구조

- @classmethod 데코레이터를 사용하여 정의
- 호출 시, 첫번째 인자로 호출하는 클래스(cls)가 전달됨 (cls 문자는 바꿔도됨)

```python
class MyClass:

		@classmethod    # 데코레이터. 이게 있어서 클래스임 없으면 인스턴스. 클래스가 되도록 기능을 붙여주는 역할
		def class_method(cls, arg1, ...):    # 데코레이터가 있어서 cls, 없으면 self
				pass
```

### 인스턴스 메서드 vs 클래스 메서드 차이

데코레이터 X - 데코레이터 O

자기 자신을 호출하는 self - 클래스 호출하는 cls

### 스태틱 메서드 (정적 메서드)

클래스와 인스턴스와 상관없이 독립적으로 동작하는 메서드

클래스와 관련이 있지만 인스턴스와 상호작용이 필요하지 않은 경우에 사용

상태와 상관이 없는 메서드

### 스태틱 메서드 구조

- @staticmethod 데코레이터를 사용하여 정의
- 호출 시 필수적으로 작성해야 할 매개변수가 없음
- 즉, 객체 상태나 클래스 상태를 수정할 수 없으며 단지 기능(행동)만을 위한 메서드로 사용

```python
class MyClass:
		
		@staticmethod
		def static_method(arg1, ...):
				pass
```

```python
{
	'caffein' : 0
	'drinkcoffe' : coffee    # 인스턴스 메서드 : 실행되면 caffein이 올라감
	'sayhello' : sayhello    # 스태틱 메서드 : 실행되어도 caffein 상태와 연관이 없음
}

def coffee():
	caffein += 10

def sayhello():
	print("hello, world!")
```

# 메서드 정리

- 인스턴스 메서드
    - 인스턴스 상태를 변경하거나, 해당 인스턴스의 특정 동작을 수행
    - self 호출
- 클래스 메서드
    - 인스턴스 상태에 의존하지 않는 기능을 정의
    - 클래스 변수를 조작하거나 클래스 레벨의 동작을 수행
    - cls 호출
- 스태틱 메서드
    - 클래스 및 인스턴스와 관련이 없는 일반적인 기능을 수행
